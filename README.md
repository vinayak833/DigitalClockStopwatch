# Digital Clock & Stopwatch

A simple and responsive Digital Clock & Stopwatch built using HTML, CSS, and JavaScript.

## Features

### Digital Clock

* Displays current time in real-time
* Updates every second
* 24-hour format
* Displays current date

### Stopwatch

* Start stopwatch
* Pause stopwatch
* Reset stopwatch
* HH:MM:SS time format

### Theme Support

* Dark Mode
* Light Mode
* One-click theme switching
* Theme preference saved using Local Storage

### User Interface

* Clean and responsive design
* User-friendly layout

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Digital-Clock-Stopwatch.git
```

2. Navigate to the project directory

```bash
cd Digital-Clock-Stopwatch
```

3. Open `index.html` in your browser
